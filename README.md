# Aura Stay Global — Luxury Hospitality Platform

A premium, full-stack responsive web application designed for luxury hospitality and hotel booking management. Built with a focus on immersive UX, dark/light contrast matrix, and fluid responsive design patterns.

## ✨ Core Features

*   **🌓 Contrast-Optimized Themes:** Fully decoupled Light & Dark modes featuring high-contrast premium typography, custom cards, and interactive overlays.
*   **📅 Quick Booking Widget:** Inline real-world booking interface module with advanced form fields and input validation engines.
*   **🖼️ Dynamic Portfolio Filtering:** Client-side room filtering system using modular control groups for zero-latency presentation transitions.
*   **📱 Responsive Breakpoint Architecture:** Fluid grid layouts tailored perfectly for mobile, tablet, and desktop viewports.
